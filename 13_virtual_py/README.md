## Step 1
- `virtualenv env_name`

## Step 2
- `.\env_name\Scripts\activate`

## Step 3
- `python --version`

## Step 4
- `pip list`

## Step 5
- `pip install pymongo`

## Step 6
- `pip list > requirement.txt`
- (if you want to install all libraries which available in requirement.txt then use this command - `pip install -r requirement.txt`)

## Step 7
- After you are created and activated a virtual environment, enter the command:
- `python -m pip install Django` (to install Django)

## Step 8
- `pip list > requirement.txt`

## Step 9
- `pip uninstall pymongo`

## Step 10
- `pip list > requirement.txt`

## Step 11
- use the command to Deactivate the virtual environment:
- `deactivate`
